## 1. SistemaCTO - Visão Geral

Frontend do sistema de gerenciamento de CTOs, desenvolvido em Angular.

A aplicação permite o cadastro e a listagem de CTOs, consumindo uma API backend desenvolvida em Spring Boot.

---

## 2. Tecnologias Utilizadas

- Angular
- TypeScript
- HTML
- SCSS
- Node.js
- NPM

---

## 3. Funcionalidades

- Cadastro de CTO
- Listagem de CTOs
- Integração com backend REST

---
